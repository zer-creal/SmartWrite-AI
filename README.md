# SmartWrite AI

## 项目简介
SmartWrite AI 是一个基于 Next.js 的在线协作写作平台，集成 Tiptap 编辑器、Supabase 认证与数据库存储，目标是提供类似 Notion 的文档编辑体验。

## 当前进度
- 编辑器基础功能已完成，包括文本输入、格式化、图片上传、代码块、链接、任务清单等。
- 已实现 BubbleMenu 工具栏、虚拟光标多人协作模拟、自动保存与文件管理（新建、删除、自动保存）。
- 基础认证系统已搭建，支持登录、注册、登出，并已集成 Supabase。
- Supabase 数据库连接成功，已配置基本 RLS 策略以保障未登录用户无法创建文档。
- 已完成编辑器样式优化，采用 Notion 风格布局及移动端适配。

## 重点待办
- 完善认证体验：增加密码重置、登录状态边界处理、页面访问保护。
- 优化文件管理：添加文件搜索、重命名和文件夹/文档分类功能。
- 优化图片上传：支持更多格式、更大文件、更稳定的上传体验。
- 添加版本控制、文档历史、导出/导入和 AI 辅助写作功能。
- 改进性能：减少页面首次加载时间、优化数据库查询、增加分页与过滤。

## 当前工作树
- `src/app/`：页面路由与入口，包括编辑器主页、文档编辑页等。
- `src/components/Editor/`：编辑器核心组件、工具栏、顶部栏与样式文件。
- `src/components/Layout/`：仪表盘布局组件。
- `src/context/`：认证与文件管理上下文实现。
- `src/hooks/`：自动保存钩子逻辑。
- `src/lib/`：Supabase 客户端与工具函数。
- `public/`：静态资源。
- `tests/`：Supabase 与文档创建测试脚本。

## 运行方式
1. 安装依赖：`pnpm install`
2. 启动开发服务器：`pnpm dev`
3. 打开浏览器访问：`http://localhost:3000`

## 额外说明
- 当前项目使用 Next.js 16.2.1 + Turbopack。
- 编辑器首次加载、动态组件与客户端路由部分需重点验证。
