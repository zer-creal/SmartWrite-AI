# SmartWrite 本地启动说明

## 项目简介
SmartWrite 是一个支持多人实时协作的智能写作平台，基于 Next.js 14 和 Tiptap 编辑器开发。

## 本地环境要求
- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器

## 启动步骤

### 1. 克隆项目
```bash
git clone <项目地址>
cd smartwrite-ai
```

### 2. 安装依赖
```bash
# 使用 npm
npm install

# 或使用 yarn
yarn install
```

### 3. 启动开发服务器
```bash
# 使用 npm
npm run dev

# 或使用 yarn
yarn dev
```

### 4. 访问应用
服务器启动后，在浏览器中打开：
- 本地地址：http://localhost:3000
- 网络地址：http://<你的IP地址>:3000

## 项目功能

### 核心功能
1. **实时协作编辑**：支持多人同时编辑文档，显示其他用户的光标位置
2. **智能悬浮工具栏**：选中文字时自动显示，提供格式化工具
3. **文档管理**：创建、浏览和管理文档列表
4. **本地数据持久化**：使用 localStorage 保存文档列表

### 技术栈
- **前端框架**：Next.js 14 (App Router)
- **编辑器**：Tiptap + React
- **样式**：Tailwind CSS
- **图标**：Lucide React
- **状态管理**：React useState + localStorage


```bash

```

```bash
npm run lint
```

## 注意事项
1. 本项目使用 localStorage 进行数据持久化，刷新页面后文档列表会保留
2. 首次启动时会自动创建 4 个默认示例文档
3. 支持响应式设计，可在移动端和桌面端正常使用
4. 悬浮工具栏会在选中文字时自动显示，跟随选区移动

## 故障排除

### 常见问题
1. **端口占用**：如果 3000 端口被占用，可在启动时指定其他端口
   ```bash
   npm run dev -- -p 3001
   ```

2. **依赖安装失败**：尝试清除 npm 缓存后重新安装
   ```bash
   npm cache clean --force
   npm install
   ```

3. **构建错误**：检查 TypeScript 类型错误
   ```bash
   npm run build
   ```
